# Employee Attendance System (C - Doubly Linked List)

A lightweight, console-based Employee Attendance Management System written in C. This project utilizes a **Doubly Linked List** data structure to manage employee records dynamically, allowing for efficient insertion, deletion, and searching.

## 🚀 Features

* **Add Employee:** Create new records with a unique ID and Name.
* **Attendance Tracking:** Mark employees as "Present" or "Absent."
* **Search & Update:** Quickly find employee details by ID or update their names.
* **Data Integrity:** Prevents duplicate IDs and handles empty list scenarios gracefully.
* **Summary Reports:** Provides a quick count of total "Present" vs. "Absent" staff.
* **Doubly Linked Navigation:** Built using `prev` and `next` pointers for flexible list traversal.

## 🛠️ Technical Specifications

* **Language:** C
* **Data Structure:** Doubly Linked List (DLL)
* **Memory Management:** Dynamic allocation via `malloc()` and `free()`.
* **Search Complexity:** $O(n)$ linear search.

## 📋 How to Run

### Prerequisites
You need a C compiler installed on your system (e.g., GCC, Clang, or MSVC).
